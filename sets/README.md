### Unique characters
Write a method name unique_chars which accepts a string as input parameters. Function returns true if string contains all unique characters except space. Else it will return False.
NOTE: Ignore the spaces
> Example 1</br>
`Input: 'abcde'`</br>
`Output: True`</br>
> Example 2</br>
`Input: 'abca'`</br>
`Output: False`</br>
